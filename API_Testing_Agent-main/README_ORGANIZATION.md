# Project Organization

## New Folder Structure

- `docs/` - All documentation files
- `testcases/` - Generated test files  
- `reports/` - Test execution reports
- `scripts/` - Utility scripts
- `output/` - Generated output files
- `src/` - Source code modules
- `templates/` - Test templates

## Where Things Are Now

**Your Agent**: `run_agent.py` (main controller)

**Generated Tests**: `testcases/test_agents_ai.py`

**Test Reports**: `reports/` folder

**Documentation**: `docs/` folder

**Scripts**: `scripts/` folder

## How to Use

```bash
python run_agent.py check    # Check setup
python run_agent.py generate # Make tests  
python run_agent.py run      # Run tests
python run_agent.py full     # Do everything
```

Done. Everything is organized now.
