================================================================================
                          TROUBLESHOOTING
================================================================================

ERROR: "Cannot connect to Ollama"
  FIX: Start Ollama server
       ollama serve

ERROR: "Model 'llama3' not found"
  FIX: Download Llama3
       ollama pull llama3

ERROR: "Request timeout"
  FIX: Wait longer (first run takes 2-5 minutes)
       Subsequent runs are faster

ERROR: "Generated code is empty"
  FIX: 1. Check Ollama is running
       2. Check Llama3 is installed
       3. Try again
       4. Check Ollama logs

================================================================================
                          TIPS & TRICKS
================================================================================

TIP 1: Keep Ollama Running
  Keep the 'ollama serve' terminal open while generating tests.

TIP 2: First Run is Slow
  Llama3 needs to load into memory (2-5 minutes).
  Subsequent runs are much faster.

TIP 3: Regenerate Anytime
  Update spec → regenerate tests
  Just re-run the command.

TIP 4: Customize Generated Tests
  Add custom test data
  Add resource-specific tests
  Extend with integration tests

TIP 5: Version Control
  Commit generated tests
  Track changes over time
  Review diffs when regenerating

TIP 6: Check Model Status
  ollama list

TIP 7: Test Model Directly
  ollama run llama3 "Hello, what is 2+2?"

TIP 8: Generate All at Once
  for resource in agents policies commissions payments; do
    python generate_tests_ai.py --resource $resource
  done
